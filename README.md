# 🏫 Hostel Management System

A full-stack hostel management web application built with **Node.js**, **Express**, and a **JSON file-based database**.

## Features
- Admin dashboard with room, student, allocation & payment management
- Student portal with login, registration & dashboard
- Reports & audit logs

## Tech Stack
- **Backend**: Node.js + Express
- **Database**: JSON file-based (no external DB required)
- **Frontend**: HTML + CSS + JavaScript

## Run Locally
```bash
npm install
node server.js
```
Visit: http://localhost:3000

**Default Admin Login:** `admin` / `admin123`
